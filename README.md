# LivinTheDreamToolkit

Repo for toolkit done on a as I need basis for Tomodachi life. 

Currently includes a c# library for reading and writing ugc textures in the save file. Also includes a gui front-end for ease of use. The gui is made in Avalonia like the save editor for cross compat purposes, (and if anyone from the save editor is reading this, feel free to just yoink all this stuff).
